# Database Files

This folder should contain the SPSS data files required for the NEXAF-AI analysis. These files are not included in the repository due to data privacy requirements.

## Required Files

| File | Description | Size |
|------|-------------|------|
| `df_ep5_fil_sep25_EFL.sav` | Episode data (170k+ AF episodes) | ~31 MB |
| `NEXAF_baselinefil_171125_EFL.sav` | Baseline characteristics (714 variables) | ~1 MB |
| `Hovedfil_analyser_hovedartikkel021225_oppdatertAFburden.sav` | Burden & outcomes (1155 variables) | ~1.3 MB |
| `valideringsdatasett300126.sav` | Validation dataset | ~80 KB |

## Variable Documentation

See **[VARIABLES.md](VARIABLES.md)** for a comprehensive list of all variables in each data file, organized by domain (demographics, comorbidities, medications, echo, CPET, outcomes, etc.).
